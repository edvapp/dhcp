# Get Info for Inventar:

we use GREP to get INFO for:

* Hosts
* Screens
* Beamers

from dhcpd.conf

Get dhcpd.conf from github.com:

>> git clone https://github.com/edvapp/dhcp.git

>> cd dhcp

grep ... 


## List all hosts:

>> grep hosts dhcpd.conf

## List all Screens:

>> grep / dhcpd.conf

## List all Beamers:

>> grep // dhcpd.conf
